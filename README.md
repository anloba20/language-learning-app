# Language Learning App

A web application for learning languages through simple games.

The idea is inspired by bilingual learning laptops: instead of studying with
plain flashcards, users practice words and phrases by playing small interactive
games. The app focuses on three languages: Russian, Estonian, and English.

## MVP Idea

- Users can create an account, log in, and log out.
- Users can choose the language they want to learn.
- Users can play language-learning games with levels.
- Users can earn points for correct answers.
- The app can save user progress.

The first game can be a simple translation challenge where the user sees a word
and chooses the correct translation from several options.
